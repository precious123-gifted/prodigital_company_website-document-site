# About

ProDigital is a dynamic e-commerce platform built with cutting-edge technologies to provide a seamless shopping experience. Leveraging the power of Next.js, TypeScript, Tailwind CSS, and other modern tools, we've crafted a robust and scalable solution.

Key Features:

Responsive Design: Enjoy a consistent and optimized experience across devices, from desktops to smartphones.
Intuitive User Interface: Navigate effortlessly through our user-friendly interface, designed with your convenience in mind.
Secure Transactions: Benefit from our secure payment gateway integration, ensuring your transactions are protected.
Product Search and Filtering: Easily find the products you're looking for with our advanced search and filtering capabilities.
Shopping Cart Functionality: Add items to your cart, customize quantities, and proceed to checkout with ease.
Order Tracking: Keep track of your orders from placement to delivery.
Technical Stack:

Frontend: React.js, Next.js, Tailwind CSS, GSAP, Prismic
Backend: Node.js, Express.js
Database: MongoDB
Cloud Storage: Cloudinary
Payment Processing: [Integrate with your payment gateway]
Other Dependencies: bcrypt, clsx, gridfs-stream, jsonwebtoken, lenis, mongoose, mongoose-gridfs, multer, multer-gridfs-storage, react-hook-form, react-hot-toast, react-image-file-resizer, react-intersection-observer, react-loading-skeleton, react-paystack, react-toastify, sass, swr, tailwind-merge
Project Structure:

frontend: Contains the React components and code for the user interface.
backend: Houses the Node.js and Express.js code for API endpoints and server-side logic.
models: Defines the data structure for products, users, and orders in MongoDB.
routes: Defines the API endpoints for various functionalities.
utils: Contains helper functions and utilities used throughout the application.
